Password complexity Checker  🎯

Overview
This project is an Ultimate Password Complexity Checker with a friendly graphical interface. It helps users evaluate the strength of their passwords and gives clear feedback to make passwords stronger and safer 🔒

Features ✨
✅ Checks for minimum length of 8 characters
✅ Ensures at least one uppercase letter
✅ Ensures at least one lowercase letter
✅ Ensures at least one digit
✅ Ensures at least one special character
✅ Displays password strength: Weak ⚠️, Moderate ⚠️, Strong 👍, Very Strong ✅
✅ Provides detailed feedback on missing criteria
✅ Easy-to-use GUI built with Tkinter 🖥️

How to Use 🚀

Make sure Python is installed and added to system PATH

Open PowerShell and navigate to the project folder

Run the program using: python password_checker_ui.py

Type your password in the input field and click Check Strength

View the password strength and feedback instantly in the GUI 💡

Files in the Project 📂
password_checker_ui.py - The main GUI Python program
README.txt - Project documentation with instructions

Requirements ⚙️
Python 3.x
Tkinter library (usually included with Python)
